# 🎙 AI Voice Interviewer

> Real human-like voice interviews powered by **Retell AI** (speech) + **Groq LLaMA 3** (free LLM) + **Claude** (evaluation)

---

## Architecture

```
Browser (mic) ──► Retell AI STT ──► WebSocket /llm-websocket ──► Groq LLaMA 3
                                                                        │
Browser (speaker) ◄── Retell AI TTS ◄──────────────────────────────────┘

Final evaluation: Claude API (claude-sonnet-4-20250514)
Proctoring:       Browser getUserMedia (camera) + getDisplayMedia (screen)
```

---

## Stack

| Layer | Tech | Cost |
|---|---|---|
| Voice STT + TTS + turn-taking | Retell AI | ~$0.07–0.10/min |

| Evaluation report | Claude (Anthropic) | ~$0.003/eval |

## Quick Start

### 1. Get API Keys

| Key | Where |
|---|---|
| `RETELL_API_KEY` | https://dashboard.retellai.com → API Keys |
| `GROQ_API_KEY` | https://console.groq.com/keys ← **FREE** |

### 2. Backend setup

```bash
cd backend
cp .env.example .env
# Fill in RETELL_API_KEY and GROQ_API_KEY

npm install
npm start
# → Server running on http://localhost:3001
```

### 3. Expose backend publicly (Retell needs a public WebSocket URL)

**Option A — ngrok (easiest for dev):**
```bash
npx ngrok http 3001
# Copy the https://xxxxx.ngrok-free.app URL
```

**Option B — Railway (free tier, for prod):**
```bash
npm install -g @railway/cli
railway login && railway init && railway up
```

**Option C — Render:**
Connect your GitHub repo → deploy as a web service (free tier works).

### 4. Create your Retell Agent (one-time)

Once your server is public, update `.env`:
```
PUBLIC_URL=https://your-public-url-here
```

Then restart the server and visit:
```
GET http://localhost:3001/setup-agent
```

This will:
- Create a Retell Custom LLM pointing at your `/llm-websocket`
- Create an agent named "Alex - AI Interviewer" with a natural voice
- Return your `agent_id` — **save it!**

Add to `.env`:
```
RETELL_AGENT_ID=agent_xxxxxxxxxxxxxxxx
```

### 5. Frontend setup

```bash
cd frontend
cp .env.example .env
# Set REACT_APP_AGENT_ID=agent_xxx (from step 4)
# Set REACT_APP_API_URL=http://localhost:3001

npm install
npm start
# → App at http://localhost:3000
```

## How It Works

### Voice Flow (per interview)

1. **Setup page**: user pastes resume + job description + agent ID → clicks Start
2. **Backend** (`POST /create-web-call`):
   - Calls Retell's `createWebCall` API with resume/JD as metadata
   - Returns an `access_token`
3. **Frontend** starts `retellClient.startCall({ accessToken })`
   - Browser mic is captured, Retell handles STT
4. **Retell** connects to your `/llm-websocket` WebSocket
5. **WS handler** receives `call_details` → creates session with resume/JD
6. On each `response_required` event:
   - Previous candidate answer is **scored by Groq** (llama-3.1-8b-instant) asynchronously
   - **Difficulty adjusts**: score ≥ 8 → harder, score ≤ 3 → easier
   - Next question generated by **Groq llama-3.3-70b** with full system prompt
   - Response **streamed back** in chunks for low latency TTS
7. After 6 questions, agent wraps up and `end_call: true` is sent
8. **Frontend** detects `call_ended` → fetches session from `/call-result/:callId`
9. **Claude** generates the final evaluation report

### Adaptive Difficulty

```
Answer score ≥ 8  → difficulty++ (harder: system design, edge cases)
Answer score 5-7  → difficulty stays
Answer score ≤ 3  → difficulty-- (simpler follow-up)
```

### Anti-Cheat Monitoring

| Signal | How |
|---|---|
| Tab switches | `visibilitychange` event |
| Window blur | `document.hidden` |
| Screen share stopped | `MediaStreamTrack ended` event |
| Camera feed | Visual display for interviewer |

All flags are POSTed to `/flag/:callId` and included in the final report.

## API Endpoints

| Method | Path | Description |
|---|---|---|
| `POST` | `/create-web-call` | Create Retell web call, returns access_token |
| `GET` | `/setup-agent` | One-time: create Retell agent pointing at this server |
| `GET` | `/call-result/:callId` | Fetch session transcript + scores after call |
| `POST` | `/flag/:callId` | Report a cheating flag from frontend |
| `GET` | `/health` | Health check |
| `WS` | `/llm-websocket` | Retell custom LLM endpoint (Groq-powered) |

## Groq Model Options

In `server.js`, swap models based on speed/quality tradeoff:

```js
// Main interviewer (quality)
model: "llama-3.3-70b-versatile"    // best quality, ~200ms
model: "llama-3.1-70b-versatile"    // also great

// Answer scorer (speed)
model: "llama-3.1-8b-instant"       // fastest, cheap
model: "gemma2-9b-it"               // good alternative
```

## Troubleshooting

**"Agent ID required"**: Run `GET /setup-agent` first and save the returned `agent_id`

**No audio**: Check browser microphone permissions

**Retell WS not connecting**: Make sure `PUBLIC_URL` uses `https://` and is reachable publicly. Retell cannot connect to `localhost`.

**Groq rate limits**: Free tier allows 30 req/min on 70b model. For high volume, switch to `llama-3.1-8b-instant`.

**Call ends immediately**: The agent wraps up after 6 questions. Change `maxQuestions` in `server.js`.
